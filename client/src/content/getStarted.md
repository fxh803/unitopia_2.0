# Get Started

Welcome to **UnitoPia** — an interactive tool for pictorial unit visualization.

## Quick Start

If you only want to design with Unitopia, use the online editor:

1. Open the homepage and click **Try Online**
2. Load a dataset or start from a blank canvas
3. Sketch markers, containers, and emitters on the canvas
4. Click **Run** to generate your pictorial unit visualization
